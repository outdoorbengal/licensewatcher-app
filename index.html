FAQ
Common questions
Everything you need to know about getting started.
Do I still need to register with each state myself? ▾
Yes — LicenseWatcher is a tracking and organization tool, not a filing service. You still submit your registrations directly with each state. But LicenseWatcher tells you exactly when, where, and how — so you never miss a deadline or scramble to find a form.
Is the state data accurate and up to date? ▾
The template comes pre-loaded with AAFCO fee data, department contacts, and deadline schedules for all 51 states. You can edit any field and add your own contacts as you build relationships with state offices. We recommend verifying fees with each state since they change occasionally.
What types of products does it support? ▾
Any animal food product that requires state-level registration: pet food, pet treats, cat supplements, dog supplements, specialty pet food, animal feed, and more. If a state requires you to register it, LicenseWatcher can track it.
How do Year 2+ renewals work? ▾
Deadlines are stored as recurring month/day patterns (e.g. "License Renewal: January 1"). As each deadline passes, the next year's instance automatically appears in your pipeline. You never have to set up Year 2 — it just rolls forward.
Where is my data stored? ▾
Your data lives in a private GitHub repository that you control. LicenseWatcher is a frontend that reads and writes to your repo via the GitHub API. You own your data completely — you can access it directly, export it, or move it anytime.
Can multiple team members use it? ▾
Yes. On the Pro plan, up to 5 team members can access the dashboard. Each person's actions are logged so you know who filed what and when. Enterprise plans support unlimited team members.
